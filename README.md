# 咖啡产业底层逻辑：一个层层妥协的生态黑盒

本项目致力于将咖啡生产处理过程中的核心数据进行开源，并利用时间戳不可篡改的特性，构建真实可靠的全链路可追溯体系。结合对行业的理解与底层思考，拆解当前咖啡全产业链中的系统性痛点。

---

### 0. 豆农：原始资源的浪费与代谢失控
* **水资源错配**：山区水资源和基础设施都是免费的，传统水洗并非精心设计的结果，水只用了它的流速做动能，冲掉的果肉变成了生物垃圾。
* **生物反应不可控**：在干燥环节，烘焙前的咖啡豆仍经历着复杂的生物反应。它是活的，但生命体的代谢物在传统日晒过程中完全处于不可控状态。

### 1. 中间商：消灭个性的“大锅饭”逻辑
* **单一维度收购**：中间商收购时不论品种、产区和质量，只在乎重量。
* **多样性抹杀**：我们农场内同时存在 Castillo、Catimor、Seni 等品种，且树龄产量各异。但在中间商逻辑下，这些差异被强行抹平，且豆农更换品种的成本极大。

### 2. 进出口商：豆子为机器服务
* **标准悖论**：10-12% 的含水率行业标准，纯粹是为了迁就脱壳机的齿轮。水分高了羊皮纸韧性强脱不了，水分低了则碎豆体。这本质上不是为了风味，而是为了出口生态的便利，为了迁就工程机械而阉割了风味上限。
* **筛选缺失**：在脱壳准备出口时没有任何有效筛选。咖啡贸易本质上仍是“发霉豆、变质豆老鼠屎坏一锅粥”的逻辑。
* **运输退化**：失去羊皮纸保护的咖啡豆在长时间海运过程中，会进一步经历回潮、氧化、酯化与破损。

### 3. 烘焙商：数据孤岛里的高级盲测
* **黑盒反推**：由于缺乏生豆的属性相关生化指标数据，烘焙师只能靠 ROR 曲线进行黑盒反推。
* **概率撞大运**：所谓的“调性”全是靠概率撞出来的。烘焙师用的生豆质量不稳定很难有效稳定产出。

### 4. 销售商：编造概念的 PPT 研发
* **概念灌输**：利用严重的信息差，将行业禁止的操作（如乱加水果发酵、添加香精）包装成“创新研发”。
* **价值倒挂**：通过刷单和品牌包装，将廉价的商业豆洗白成精品，获取超额溢价。

---

## 核心总结

这个链条中层层加码，建立起了**低效且自保的护城河**，不断增加系统的信息差与熵。

如果不打破这种**“机械和利益优先”**的逻辑，精品咖啡永远只能是“玄学”和“草台班子”的循环。在这种固化的生态下，**质量稳定和风味不再是第一性原理**，取而代之的是对行业旧生态不断的妥协。



为了保护核心生化模型与自动化算法，本项目仅开源接口定义（Header Files）。所有核心逻辑的演进均已通过带有时间戳的加密 Commit 记录，确保开发历程的真实性与可追溯性。
